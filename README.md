# hubot-retty

A hubot script that does the things

See [`src/retty.coffee`](src/retty.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-retty --save`

Then add **hubot-retty** to your `external-scripts.json`:

```json
[
  "hubot-retty"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
